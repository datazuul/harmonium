Provides a sample implementation of the HME hosting API.

Running an application

  • To launch an HME application, run the Main class, and provide the application class name as an argument
  • The Main class is called com.tivo.hme.host.sample.Main. 
    % java -classpath hme.jar:hme-host-sample.jar:samples.jar \
com.tivo.hme.host.sample.Main com.tivo.hme.samples.hello.HelloWorld
